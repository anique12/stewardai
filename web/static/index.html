StewardAI — Voice Core

Browser test pages for the cascaded voice agent (STT → LLM → TTS). Audio: PCM s16le, 16 kHz, mono. Running with the configured backends.

Test pages

TTS  —  text to speech
Type text, pick a voice, synthesize a WAV and play it back.
STT  —  speech to text
Record from your mic; the endpointer cuts utterances and the transcript appears live.
Pipeline  —  full voice loop
Talk → transcript → streamed LLM reply → streamed audio back, with a per-stage timing panel.
